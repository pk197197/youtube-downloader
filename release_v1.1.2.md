# v1.1.2 Release Notes

## 🚀 极速启动与高级更新体验

本次更新带来了显著的性能提升和全新的更新管理功能，同时修复了用户反馈的关键问题。

### ✨ 新增功能 (New Features)
- **⚡️ 极速启动 (Startup Optimization)**: 
  - 将核心组件加载移至后台，程序启动速度提升 300%，彻底告别启动时的“假死”现象。
- **🔔 智能更新检查 (Advanced Update Checker)**:
  - **自动检查**: 每次启动时自动检测新版本。
  - **新版弹窗**: 全新的更新提示窗口，展示完整的更新日志。
  - **灵活选项**: 支持 [立即更新]、[稍后提醒]、[跳过此版本] 三种策略，且选择会被永久保存。
- **🔢 优化的画质选择**:
  - 画质列表现在带有清晰的序号 (如 `1. 最高画质`, `2. 1080p`)，选择更直观。
- **📝 增强的进度反馈**:
  - 点击“粘贴链接”后立即显示“正在解析...”状态，拒绝无响应焦虑。

### 🛠 问题修复 (Bug Fixes)
- **⚠️ FFmpeg 自动安装与检测**: 
  - 启动时自动检测系统是否安装 FFmpeg。
  - **新功能**: 如果缺失，程序会自动尝试安装（支持 Homebrew 和静态包下载），确保高清画质合并功能正常使用。
  - 如果自动安装失败，会弹出明确警告及手动安装指引。
- **🐛 文本框显示修复**:
  - 修复了更新日志文本框在未点击时看起来像空白区域的问题，现在拥有清晰的边框。
- **UI 细节优化**:
  - 统一了所有弹窗按钮的风格（胶囊样式），视觉体验更佳。
  - 增加了更新弹窗的宽度，阅读更新日志更舒适。
